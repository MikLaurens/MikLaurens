Для дальнейшей работы с данным проектом Вам нужно установить библиотеку pyqt5 и sqlite3
Весь код проекта содержится в файлах:
main
database
reports
ui

Создадите проект, включите эти файлы в его список, запустите main и он будет работать без exe. Далее можете редактировать его по
своему желанию.
